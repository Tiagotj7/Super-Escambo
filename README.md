# 🛒 Super Escambo

Projeto de interface web desenvolvido como **atividade do curso Técnico do SENAI**, com foco em **UI/UX, responsividade e manipulação de DOM com JavaScript**.

O sistema simula uma **loja virtual simples**, contendo catálogo de produtos, ofertas, carrinho e alternância de tema (modo claro e escuro).

---

# 📚 Sobre o Projeto

O **Super Escambo** é uma aplicação web estática desenvolvida utilizando:

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla)**

O objetivo do projeto é demonstrar conhecimentos em:

* Estruturação de páginas web
* Design responsivo
* Interface amigável ao usuário (UI/UX)
* Manipulação de elementos com JavaScript
* Implementação de tema **Light/Dark**
* Uso de **sidebar responsiva**
* Interações dinâmicas na página

---

# 🎯 Objetivos da Atividade

Esta atividade tem como finalidade:

* Praticar **desenvolvimento front-end**
* Criar uma **interface moderna e responsiva**
* Aplicar conceitos de **usabilidade**
* Implementar **interações com JavaScript**
* Desenvolver organização de código em **HTML, CSS e JS**

---

# 🖥️ Funcionalidades

O projeto possui as seguintes funcionalidades:

### 📦 Catálogo de Produtos

* Exibição de produtos em **cards**
* Nome do produto
* Imagem
* Preço
* Botão de compra

### 🛍️ Carrinho de Compras

* Contador de itens adicionados
* Atualização dinâmica com JavaScript

### 🔥 Seção de Ofertas

* Produtos em destaque
* Layout semelhante ao catálogo principal

### ✉️ Formulário de Contato

* Campo de nome
* Campo de email
* Campo de mensagem
* Botões de envio e limpeza

### 🌙 Tema Claro / Escuro

* Alternância entre **modo light e dark**
* Ícones de **sol ☀️ e lua 🌙**
* Preferência salva no **localStorage**

### 📱 Responsividade

O layout foi desenvolvido para funcionar em:

* 📱 Celulares
* 💻 Computadores
* 📟 Tablets

Utilizando **Grid Layout e Media Queries**.

### 📂 Sidebar Responsiva

Menu lateral contendo:

* Início
* Produtos
* Ofertas
* Contato

A sidebar pode ser aberta pelo botão **☰** no header.
